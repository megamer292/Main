SUMMARY

The main purpose of this file/site is to hide games sites from blocksi, as due to the file being under a file// url, most in-window content cannot be accessed by the extension. Any updates will roll out slowly, but the process can be sped up by clearing browser cache.

DISPLAYING SITES

The first input box is for providing URLs to the site you want displayed. This does not work for any site, only those which allow Iframe embedding.
